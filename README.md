# StockQuest Simulator

A level-based stock trading simulation game where users make decisions to grow their wealth through buying and selling stocks.

## 🚀 Features
- Buy and sell stocks  
- Multiple levels with different scenarios  
- Profit percentage calculation  
- Portfolio tracking  
- Transaction history  

## 🛠 Tech Stack
- Java (Swing)  
- MySQL  

## 📂 Project Structure
- `StockQuestGaming.java` → Main application  
- `database.sql` → Database schema and sample data  

## ⚙️ How to Run
1. Clone or download the repository  
2. Open the project in NetBeans or any Java IDE  
3. Import and run `database.sql` in MySQL  
4. Update database username and password in the code  
5. Run the main class  

## 🧠 Concept
This project simulates real-world stock trading decisions using scenarios and volatility, helping users understand risk and profit strategies.

## 📌 Note
- This is a learning project  
- Database credentials should be configured locally
